# 蓝海-智医云诊

## 系统介绍：

《蓝海-云上医视频问诊》是一款医疗服务平台。

旨在提供便捷、安全、高效的医疗服务。

患者可通过在线挂号与医生远程视频问诊。

### 前端：

### 后端：

+ SpringBoot 2.7
+ Minio
+ Sa-Token
+ Redis
+ Docker
+ Quartz
